# NLP Project


## Structure
- **src/**: Source code (.c, .py, .scala, etc.) chứa mã nguồn của các lab 01, 02, 04, 05

- **report/**: Project reports chứa báo cáo của tất cả các lab

- **notebook/**: Jupyter notebooks chứa các notebook thí nghiệm của một số lab 04, 06, 07

- **test/**: Test scripts, chứa các script kiểm thử của các lab 01, 04, 05

- **data/**: Datasets, chứa một phần nhỏ dữ liệu của file gốc ban đầu
